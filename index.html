About CSS

CSS (Cascading Style Sheets) is used to style and format HTML documents. It controls colors, fonts, layouts, and spacing.

The Box Model

Every HTML element can be thought of as a box, consisting of:

  • Content – The text or image inside
  • Padding – Space between content and border
  • Border – The edge around the element
  • Margin – Space outside the border
